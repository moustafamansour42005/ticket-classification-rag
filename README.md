# 🎫 Ticket Classification using RAG

An intelligent ticket classification system built using Retrieval-Augmented Generation (RAG).

## 🚀 Project Overview
This project classifies customer support tickets using:
- Embeddings
- Vector Database
- Large Language Models (LLMs)
- Retrieval-Augmented Generation (RAG)

## 🧠 How It Works
1. Convert tickets into embeddings
2. Store embeddings in vector database
3. Retrieve similar historical tickets
4. Use LLM to classify the new ticket based on retrieved context

## 🛠 Tech Stack
- Python
- LangChain / LlamaIndex
- OpenAI / LLM API
- FAISS / Pinecone
- Pandas / NumPy

## 📌 Use Cases
- Customer Support Automation
- IT Helpdesk Classification
- Smart Email Routing
